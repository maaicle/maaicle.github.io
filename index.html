HK Drywall Bid Calculator

For my first personal project I built a website for my Dad's business. The front page displays basic HTML/CSS markup as well as some advanced javascript such as IntersectionObserver.

The bid calculator uses axios calls to my back-end to view, create, update, and delete invoice items. I used express for my back end and tied it to a postgres database. The app was deployed on Heroku.com
